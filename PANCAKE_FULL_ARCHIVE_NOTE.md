## Competition format

- Kaggle slug: `CayleyPy-pancake` / runner key `cayleypy-pancake`
- `test.csv` columns: `id,n,permutation`
- `submission.csv` columns: `id,permutation,solution`
- Move `Rk`: reverse the first `k` elements of the permutation.
- `solution`: dot-separated moves, e.g. `R4.R2`.
- Objective: minimize total number of moves.
