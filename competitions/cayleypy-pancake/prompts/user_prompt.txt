We are solving the CayleyPy Pancake sorting Kaggle competition.
Reference: https://www.kaggle.com/competitions/CayleyPy-pancake

TASK
Write solve_module.py that sorts a permutation using ONLY prefix-reversal moves.

ALLOWED MOVE
- Rk: reverse the first k elements, where 2 <= k <= n.

INPUT / OUTPUT CONTRACT
- Implement solve(vec) -> (moves, sorted_array)
- vec is a Python list[int]
- moves must be a list[str] containing only tokens like "R5", or the string "UNSOLVED"
- sorted_array must be the final array after applying the moves
- Script mode must print JSON: {"moves": ..., "sorted_array": ...}

ALGORITHM REQUIREMENTS
- Polynomial time only.
- No BFS / DFS / A* / exhaustive search.
- Do not use slicing to perform a flip; implement the reversal with a two-pointer swap loop.
- Do not use sorted() or list.sort() for the core sorting logic.

RECOMMENDED IMPLEMENTATION ORDER
1. If bundled test.csv + sample_submission.csv exist, you may first build an exact lookup baseline:
       initial_state/permutation string -> valid path
   Return that exact path for exact matches.
2. For all other vectors, implement classic Pancake Sort:
   for size = n, n-1, ..., 2:
     find the index of the maximum value inside vec[0:size]
     if it is not already at size-1:
       if index != 0: apply flip(index+1)
       apply flip(size)
3. This gives a constructive O(n^2) algorithm and is fully valid.

IMPLEMENTATION DETAILS
- Keep a copy of vec and mutate it in place.
- Append every move token immediately after performing it.
- Verify that sorted_array is exactly the result of the emitted moves.
- Return exactly one complete Python file inside one ```python``` block.
- Do not add explanations outside the code block.
