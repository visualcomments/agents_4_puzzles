We are solving the CayleyPy Reversals Kaggle competition.
Reference: https://www.kaggle.com/competitions/cayleypy-reversals

TASK
Write solve_module.py that sorts a permutation using ONLY reversal moves.

ALLOWED MOVE
- R[i,j]: reverse the contiguous segment vec[i..j] inclusive.

INPUT / OUTPUT CONTRACT
- Implement solve(vec) -> (moves, sorted_array)
- moves must be list[str] with tokens in the exact format "R[i,j]", or the string "UNSOLVED"
- sorted_array must be the final array after applying the moves
- Script mode must print JSON: {"moves": ..., "sorted_array": ...}

ALGORITHM REQUIREMENTS
- Polynomial time only.
- No BFS / DFS / A* / exhaustive search.
- Do not use slicing for the reversal; use a two-pointer swap loop.
- Do not use sorted() or list.sort() for the core algorithm.

RECOMMENDED IMPLEMENTATION ORDER
1. If bundled test.csv + sample_submission.csv exist, you may first build an exact lookup baseline:
       initial_state/permutation string -> valid path
   Return that exact path for exact matches.
2. For all other vectors, implement the direct constructive algorithm:
   for i from 0 to n-1:
     find idx where vec[idx] == i
     if idx != i: apply R[i,idx]
   This places each value i into position i and runs in O(n^2).

IMPLEMENTATION DETAILS
- Work on a copy of vec.
- Append each move immediately after performing it.
- Verify that sorted_array equals the state produced by the moves.
- Return exactly one complete Python file inside one ```python``` block.
- Do not add explanations outside the code block.
